Installation

Prêt en
30 secondes.

npm, CDN ou import direct — choisis la méthode qui te convient.

$ npm install pulsa-css

// Dans ton fichier CSS ou SCSS :
@import 'pulsa-css';

// Ou import partiel (recommandé) :
@import 'pulsa-css/pulsa-tokens.css';
@import 'pulsa-css/pulsa-base.css';
@import 'pulsa-css/pulsa-components.css';
package.json → pulsa-css@2.2.0
<!-- Import complet via CDN -->
<link rel="stylesheet"
     href="https://unpkg.com/pulsa-css@2/pulsa.css"/>

<!-- Version spécifique -->
<link rel="stylesheet"
     href="https://cdn.jsdelivr.net/npm/pulsa-css@2.2.0/pulsa.css"/>
CDN — unpkg.com / jsdelivr.net
<!DOCTYPE html>
<html data-theme="dark" data-accent="rose">
<head>
<link rel="stylesheet" href="pulsa.css"/>
</head>
<body>
<button class="pulsa-btn-primary">Hello Pulsa !</button>
</body>
</html>
index.html — configuration minimale
// Theme toggle basique
const btn = document.querySelector('#themeBtn');
btn.addEventListener('click', () => {
const html = document.documentElement;
html.dataset.theme = html.dataset.theme === 'dark' ? 'light' : 'dark';
});

// Scroll reveal avec Intersection Observer
new IntersectionObserver((entries) => {
entries.forEach(e => e.target.classList.toggle('revealed', e.isIntersecting));
}).observe(document.querySelectorAll('[data-pulsa-reveal]'));
